# MASGlobal

MasGlobal Web Test es el proyecto que les presento par cumplir el test que me enviaron.

### Para tener en cuenta

- el proyecto web normal se puede ver en https://localhost:portnumber/
- la documentación del web api propio lo pueden ver en https://localhost:portnumber/swagger
- Para la ejecución del api:
  - _Info de un solo empleado_: https://localhost:44373/api/Employee/1
  - _Info de todos los empleados_: https://localhost:44373/api/Employee/All
